# Kali Linux Cheat Sheets 🐱‍💻

A sleek and simple HTML/CSS website offering quick-reference **Kali Linux cheat sheets** designed for ethical hackers, penetration testers, and cybersecurity enthusiasts. Hosted on GitHub Pages for easy access and sharing.

## 🔥 Features

- 💡 **Quick Command Reference** for common tasks in Kali Linux
- 🎨 Eye-catching **animated color effects** using pure CSS
- 📷 Visual layout with cheat sheet image
- 📥 **Downloadable PDF** for offline access
- 👣 Footer with GitHub & LinkedIn social links
